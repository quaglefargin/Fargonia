# Fargonia

Fargonia is a dark fantasy simulation sandbox focused on embodied play, autonomous NPC agency, social pressure, rumor propagation, combat, memory, and counterplay.

## Project Goals

- Build a world that lives without the player
- Keep systems readable enough to remain fun
- Support embodied play and observer mode
- Unify combat, social, spiritual, environmental, and meta adversity through shared learning architecture

## Project Structure

- `src/core/` — engine backbone
- `src/ui/` — terminal input and presentation
- `src/world/` — pressure, settlement, and world logic
- `src/social/` — relationships, rumors, and social fallout
- `src/combat/` — combat systems and tactical logic
- `src/counterplay/` — lessons, counters, and readiness
- `src/npcs/` — autonomous NPC initiative
- `src/observer/` — observer mode and simulation watching
- `src/persistence/` — save/load and schema repair
- `docs/` — doctrine, schema, setup, and design references
- `scripts/` — smoke tests and helper tools

## Status

This repository is being scaffolded from scratch with a professional, teachable structure.
